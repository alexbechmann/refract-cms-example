# Refract-Cms Example

# Prequisites

- Node
- Docker

# Clone

```
git clone https://github.com/alexbechmann/refract-cms-example.git
cd refract-cms-example
```

## Develop

```
yarn
yarn start
```

Or if you don't have yarn installed

```
npx yarn
npx yarn start
```
